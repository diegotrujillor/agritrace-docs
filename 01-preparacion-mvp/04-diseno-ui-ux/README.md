# Diseño UI/UX

Especificaciones completas de diseño de interfaz y experiencia de usuario, incluyendo directrices, especificaciones de pantallas, flujos de usuario, marca y prototipo interactivo.

## Orden de Lectura

Sigue esta secuencia para comprender el alcance completo del diseño:

1. **[01-directrices-ui-ux](01-directrices-ui-ux/)** - Principios y directrices de diseño fundamentales
2. **[02-especificaciones-pantallas](02-especificaciones-pantallas/)** - Especificaciones detalladas de cada pantalla
3. **[03-mapas-recorrido-usuario](03-mapas-recorrido-usuario/)** - Experiencias de usuario y journey maps
4. **[04-diseno-plataforma](04-diseno-plataforma/)** - Marca, paleta de colores e iconografía
5. **[05-guia-prototipo](05-guia-prototipo/)** - Guía del prototipo interactivo en Figma

### [02-especificaciones-pantallas](02-especificaciones-pantallas/)

- ✅ Directrices y principios de diseño documentados

- ✅ Journey maps y análisis de usuario completados

- ✅ Prototipo interactivo en Figma validado
